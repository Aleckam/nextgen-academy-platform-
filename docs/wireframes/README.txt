CONTENTS
--------
This package contains 4 wireframes covering the priority
screens identified for the NextGen Academy MVP.

WF01 — School Admin Dashboard
  Product face : School Portal
  Device       : Desktop
  Description  : The main screen a headmaster sees on login.
                 Shows enrolled students, completion rates,
                 certificates earned, inactive student alerts,
                 and the auto-generated term report download.

WF02 — Individual Class View
  Product face : School Portal
  Device       : Desktop
  Description  : Drill-down into a single class (Form 2B).
                 Module progress strip, per-student progress
                 table with quiz scores and last-active dates,
                 and a live activity feed. Teacher view is
                 read-only — no marking required.

WF03 — CSV Student Upload + PIN Card
  Product face : School Portal
  Device       : Desktop
  Description  : The student onboarding flow. Admin uploads a
                 CSV file (First name, Last name, Class name).
                 Platform validates, auto-generates 4-digit PINs,
                 and produces a printable PIN card PDF per class.
                 No email addresses required for students.

WF04 — Young Investors Lesson Player
  Product face : Learner Application
  Device       : Desktop (Ages 13–18)
  Description  : The core student content experience. Video
                 player with progress controls, breadcrumb nav,
                 lesson title with ZSE/VFEX badges, key terms
                 panel (Zimbabwe-contextualised), quiz unlock
                 prompt with coin reward, module lesson list
                 sidebar, virtual ZSE/VFEX portfolio widget,
                 and achievement badges.
